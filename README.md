# RenameProg

Пакетная утилита для переименования чертежей ЕСКД и правки обозначений внутри PDF.

`rename_files.py` работает в своей рабочей папке и по правилам замены из `rename_args.txt`:

1. **Переименовывает файлы** — подставляет фрагменты в имена файлов.
2. **Правит содержимое PDF** (текстовых, не сканов) — заменяет те же подстроки прямо в документе: обычно это код обозначения в основной надписи (штамп, правый нижний угол) и его повёрнутая копия в графе 26 (верхний левый угол).

Перед любыми изменениями утилита показывает полный план и запрашивает подтверждение.

## Установка

Нужен Python 3.10+ и `pymupdf`:

```powershell
python -m venv .venv
.venv\Scripts\pip.exe install pymupdf
```

## Запуск

```powershell
# Как скрипт
.venv\Scripts\python.exe rename_files.py

# Сборка standalone .exe (Windows)
.venv\Scripts\pyinstaller.exe --onefile rename_files.py
```

При запуске из `.exe` файл `rename_args.txt` должен лежать рядом с `.exe`.

## Формат `rename_args.txt`

```
# строки-комментарии игнорируются
В671 to В345
старый_фрагмент to новый_фрагмент
```

Одно правило на строку: `<подстрока> to <подстрока>`. Каждое правило применяется
независимо к каждому имени файла и, для `.pdf`, к каждой странице содержимого.

### Настройки

Помимо правил замены, файл может содержать строки-настройки вида `ключ = значение`:

- **`apply_pdf_replacement`** (по умолчанию `True`) — если задать `False`, утилита только
  переименовывает файлы, не трогая содержимое PDF.

Пример — только переименование, без правки штампов:

```
apply_pdf_replacement = False
В671 to В345
```

## Как это работает

- **Замена текста в PDF** идёт в два шага (`apply_pdf_replacements`):
  1. Старый текст стирается через `add_redact_annot(fill=(1,1,1))` + `apply_redactions(images=0, graphics=0)` — линии и изображения не затрагиваются.
  2. Новый текст вставляется в той же позиции (`insert_text`) шрифтом ГОСТ тип А
     (`C:\Windows\Fonts\GOST_AU.ttf`), так как встроенные base14-шрифты PyMuPDF не
     поддерживают кириллицу. Исходный шрифт чертежа не сохраняется — заменённые
     фрагменты отрисовываются в ГОСТ тип А курсив.
  3. Размер шрифта определяется по ближайшему текстовому спану, по умолчанию 10pt.
- **Атомарное сохранение**: PDF пишется во временный файл `<path>.tmp`, затем
  заменяет оригинал через `os.replace`.
- Сами `rename_args.txt` и скрипт/`.exe` из обработки исключаются.

## Состав

- `rename_files.py` — основная утилита.
- `rename_args.txt` — правила замены.
- `smoke_test.py` — базовая проверка.
